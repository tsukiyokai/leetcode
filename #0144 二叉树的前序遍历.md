## 解法一

### 思路

```
递归
```

### 代码

```c++
class Solution {
public:
    vector<int> preorderTraversal(TreeNode* root) {
        vector<int> res;
        fun(root, res);
        return res;
    }

    void fun(TreeNode* root, vector<int>& a)
    {
        if (root == NULL) return;
        a.push_back(root->val);
        fun(root->left, a);
        fun(root->right, a);
    }
};
```

### 注释

```
返回值类型：vector<int>
函数名：preorderTraversal
传入参数：根节点
初始化int类型的容器res，
向函数fun传入根节点root和res
返回res。

返回值类型：void
函数名：fun
传入参数：根节点，容器res的引用a  // 因为要修改容器res的内容，所以不能按传值方式，要传引用。
如果传入结点为空，无需遍历，直接返回；
否则传入结点非空，则：
访问当前结点，将其数据域压入容器res的引用a；
按前序遍历递归地访问当前节点的左子树；
按前序遍历递归地访问当前节点的右子树；
```

## 解法二

### 思路

```
迭代
```

### 代码

```c++
class Solution {
public:
    vector<int> preorderTraversal(TreeNode* root) {
        vector<int> res;
        fun(root, res);
        return res;
    }

    void fun(TreeNode* root, vector<int>& a) {
        stack<TreeNode*> s;
        TreeNode* curr = root;
        if (root == nullptr) return;
        while (curr != nullptr || s.empty() == false) {
            while (curr != nullptr) {
                a.push_back(curr->val);
                if (curr->right != nullptr) s.push(curr->right);
                curr = curr->left;
            }
            if (!s.empty()) {
                curr = s.top();
                s.pop();
            }
        }
    }
};
```

### 注释

```
返回值类型：vector<int>
函数名：preorderTraversal
传入参数：根节点
初始化int类型的容器res，
向函数fun传入根节点root和res
返回res。

返回值类型：void
函数名：fun
传入参数：根节点，容器res的引用a  // 因为要修改容器res的内容，所以不能按传值方式，要传引用。
初始化一个辅助容器，名为s的栈，栈中的每个元素的类型均为指向TreeNode类型的指针；
声明一个指向当前结点的指针curr，将其初始化为指向根结点；
当（栈s非空 或 当前结点curr非空）时，进行循环：
    当（当前结点curr非空）时，进行循环：
        （访问根）向容器res的引用a中压入当前结点的数据域； // 向左迭代过程中，根和左都能依次访问，而右则暂时存入辅助栈。
        如果（当前结点curr的右邻域非空），则将curr的右结点压入辅助栈s中；
        curr向左移动；
    上面循环结束时，curr指向最左边结点的左邻域，为空；// 循环结束时根左都访问完成，之后依次弹出栈中事先存放的右结点们即可。
    如果（辅助栈s非空），则：
        令curr=辅助栈s的顶部元素；
        s弹出一次；
```
