## 题目

```
罗马数字包含以下七种字符: I，V，X，L，C，D，M。

字符          数值
I             1
V             5
X             10
L             50
C             100
D             500
M             1000

例如，罗马数字2写做II，即为两个并列的I。12写做XII，即为X+II。27写做XXVII, 即为XX+V+II 。
通常情况下，罗马数字中小的数字在大的数字的右边。
给定一个罗马数字，将其转换成整数。输入确保在1到3999的范围内。
```

## 解法

### 思路

```
哈希
```

### 代码

```c++
class Solution {
public:
    int romanToInt(string s) {
        map<char, int> m = {
            {'I',1} ,
            {'V', 5},
            {'X', 10},
            {'L', 50},
            {'C', 100},
            {'D', 500},
            {'M', 1000} };

        int res = 0;
        for (unsigned i = 0; i < s.length(); ++i) {
            m[s[i]] < m[s[i + 1]] ? res -= m[s[i]] : res += m[s[i]];
        }
        return res;
    }
};
```

### 注释

```
注意理解规则：
处理某一位上的数字的时候，如果它小于右手边的数字，对它就使用减法；
处理某一位上的数字的时候，如果它大于右手边的数字，对它就使用加法。
```
