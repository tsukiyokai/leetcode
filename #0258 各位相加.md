## 题目

```
给定一个非负整数 num，反复将各个位上的数字相加，直到结果为一位数。
```

## 解法

### 代码

```c++
class Solution {
public:
    int addDigits(int num) {
        while (num > 9) {
            num = sumDigits(num);
        }
        return num;
    }

    int sumDigits(int num) {
        int sum0 = 0;
        string str = to_string(num);
        for (auto it : str) {
            sum0 += it - '0';
        }
        return sum0;
    }
};
```
